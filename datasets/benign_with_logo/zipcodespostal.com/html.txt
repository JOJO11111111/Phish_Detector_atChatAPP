<h2>Zip codes</h2>
<p>Zip code can be <a href="https://en.wikipedia.org/wiki/ZIP_Code">defined as</a> a system used in the united state by the <a href="https://m.usps.com/m/ZipLookupAction">United state postal service(USPS)</a> to enhance the delivery of mail. The word zip is an acronym for <strong>Zone Improvement Plan.</strong><br/>
zip codes or postal codes are widely used for routing mails, sorting packages and calculating the time and cost of shipping a package and so on.</p>
<h3>What is the differences between zip codes and postal codes?</h3>
<p><span class="oneClick-link">people seems to be confused of this two words, but the truth is that they serve as the same purpose. The only different is that zip code is mainly used in the US while postcode or postal codes are used in other countries. Mind you, not all countries uses zip codes or postal codes.</span></p>
<p>Zip codes/ postal code format in some countries:<br/>
USA: CA 90026 = somewhere in Los Angeles, California<br/>
UK: NW2 1QA = somewhere in northwest London, UK<br/>
China: 999077 = official postcode for Hong Kong<br/>
Japan: 103 = somewhere in Tokyo, Japan</p>

There are four types of ZIP Codes:<p></p>
<h3>Types of Zip Codes</h3>
<ol>
<li>Unique address: Big organisations like Hospitals, universities, Government agencies and businesses that receives a lot of mails uses unique zip codes.</li>
<li>PO Box only</li>
<li>Military</li>
<li>Standard (all other ZIP codes).</li>
</ol>
<h2>Uses of Zip codes/Postal codes</h2>
<ol>
<li>Mail services: this is the main use of zip codes and postal codes.</li>
<li>Delivery services:used for internal routing of a package.</li>
<li>Statistics: For example, in the United State, there are over 42000 zip codes. Apart from delivery mails, zip codes are also used in gathering statistics.</li>
<li>Marketing(Zip-codes marketing)</li>
<li>Legislative districts</li>
<li>Internet purposes: You will often find zip code under most education or eCommerce website registration page.</li>
<li>Credit card securityip codes can also be used for credit card authorization.</li>
</ol>
<p>Insurance rating. some insurance carriers uses zip codes to determine the rate the customer will receive.</p>
