<h1><a href="/about/">Users</a></h1>
<h2>A powerful, free design tool</h2>
<p>Whether you are an illustrator, designer, web designer or just someone who needs to create some vector imagery, Inkscape is for you!</p>
<ul class="ticks">
<li>Flexible drawing tools</li>
<li>Broad file format compatibility</li>
<li>Powerful text tool</li>
<li>Bezier and spiro curves</li>
</ul>
<p>Want to find out more about how Inkscape can help you? Look at the full set of <a href="/about/features/">features</a> or <a href="/release/">try it</a>!</p>

<p>The Inkscape project has released a beta version of the upcoming and much-awaited Inkscape 1.0 on September 17th!</p>
